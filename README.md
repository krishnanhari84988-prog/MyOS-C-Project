# MyOS - Mini Operating System in C

## 📌 Description

This project is a simple Operating System simulator built using C.
It provides a command-line interface with login authentication.

## 🚀 Features

* 🔐 Login system (username & password)
* 💻 Command-line interface
* 🧾 Commands:

  * help
  * clear
  * about
  * time
  * date
  * exit

## 🛠 Technologies Used

* C Programming
* GCC Compiler
* MSYS2
* Visual Studio Code

## ▶️ How to Run

gcc main.c login.c commands.c utils.c -o myos.exe
./myos.exe
